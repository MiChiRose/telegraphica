# 🏞️ Telegraphica v0.4.9 ✦ Open Beta

Telegraphica v0.4.9 continues the open beta line for OS X 10.9.5 Mavericks on Intel x86_64. This release focuses on chat polish, safer message handling, search/navigation work, and a much larger set of visual themes.

## What’s New

- 🔍 Improved chat search and search navigation foundations.
- 💬 Better reply, edit, delete, comments, and message action handling.
- 📌 More reliable pinned-message presentation.
- 📨 Better handling for long outgoing and displayed messages.
- 🖼 Media improvements for images, albums, unsupported files, and large-file paths.
- 🧽 Cache cleanup and economy/autodownload settings polish.
- 🎨 New Retro and Experimental theme families, including Frutiger-style, Y2K, Metro Pop, and Matrix-inspired looks.
- 🧾 Translation and UI-label cleanup across chat headers, folders, pinned messages, and settings.
- 🛠 Additional defensive fallbacks so individual components fail more softly instead of crashing the app.

## Fixes

- Fixed unreadable Settings switch labels in dark experimental themes.
- Fixed contrast regressions in layered/dark theme cards.
- Fixed theme text contrast regressions that affected some dark and retro themes.
- Fixed read/check marker sizing consistency across message types.
- Fixed update badge and settings notification polish.
- Fixed self-chat presentation as Favorites.

## Notes

Telegraphica is still an experimental, unofficial Telegram client. Release confidence for this build comes from the OS X 10.9.5 / Xcode 6.2 legacy lane. The DMG must include the bundled Mavericks-compatible `libtdjson.dylib`; otherwise it is not an end-user installer.
