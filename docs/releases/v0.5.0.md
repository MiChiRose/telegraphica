# 🌟 Telegraphica v0.5.0 ✦ Open Beta

Telegraphica v0.5.0 is the big open beta milestone for OS X 10.9.5 Mavericks on Intel x86_64. This release turns the app from a promising experiment into a much more complete daily messaging client: richer chat info, better media browsing, sticker workflows, visual themes, update polish, and a long list of defensive legacy-Mac fixes.

## What’s New

- 🧭 New Chat Info window with profile summary, themed layout, media/files/links/music/voice/GIF/polls/groups sections, preview opening, and paged media loading.
- 🖼 Better media browsing: newer items first, image preview instead of surprise jumps, “go to message” through context actions, and safer loading states.
- 🧩 Reworked sticker picker with recent/favorite controls, horizontal sticker-pack rail, pack preview preloading, cleaner icon buttons, and Mavericks-safe scroll behavior.
- 🎨 New Visual Worlds theme category plus more retro/experimental themes, including Frutiger and Y2K-inspired variants.
- 💬 Closed-chat navigation: Escape can close the active dialog and show recent personal chats instead of leaving a blank, confusing pane.
- 📎 Channel/about links now understand Telegraphica’s own channel flow and can open the Telegraphica channel inside the app.
- 🧰 More settings and display preferences, including chat display tuning and message block presentation work.
- 🔍 More complete chat search and media/search navigation foundations.
- 🧪 Additional unit/static tests for media pagination, legacy compatibility, localization, theme/resource policy, and core layout behavior.

## Fixes

- Fixed hidden sticker-pack rail scrolling and corrected horizontal trackpad direction.
- Fixed Chat Info media pagination so scrolling to the bottom can load more items.
- Fixed Mavericks crash paths around media center/search-field APIs.
- Fixed active-session selection regressions and device termination UI behavior.
- Fixed several dark/retro theme contrast regressions.
- Fixed settings scroll/layout regressions and bottom Help/update area visibility.
- Fixed message block layout for media, reactions, and comments.
- Fixed update/installer flow details and close-window behavior.
- Fixed paste/keyboard shortcut regressions that could accidentally replace typed text.
- Fixed self-chat/Favorites presentation and folder/drawer icon polish.

## Notes

Telegraphica remains an experimental, unofficial Telegram client. Release confidence for this build comes from the OS X 10.9.5 / Xcode 6.2 legacy lane and repeated HITL checks on real Mavericks hardware.

The DMG must include the bundled Mavericks-compatible `libtdjson.dylib`; otherwise it is not an end-user installer.
