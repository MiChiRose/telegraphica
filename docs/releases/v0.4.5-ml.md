# Telegraphica 0.4.5(ml) — Mountain Lion Special Build

Telegraphica `0.4.5-ml` is a special compatibility release for **OS X 10.8.5
Mountain Lion** on Intel `x86_64`. It is built from the separate
`mountain-lion/*` branch line and is intentionally independent from the regular
Mavericks `main` / `develop` release line.

## What Should I Download?

If you are testing Mountain Lion, download this file from **Assets**:

**`Telegraphica-v0.4.5-ml-macos10.8-x86_64.dmg`**

Then open the DMG and drag **Telegraphica.app** into **Applications**.

The `.app.zip` file is an alternate app bundle package. The `.sha256` files are
for checksum verification.

## Mountain Lion Notes

- Special release for OS X 10.8.5 Mountain Lion only.
- Built with Xcode 5.1.1 and the OS X 10.9 SDK, targeting OS X 10.8.
- Uses a Mountain Lion-compatible TDLib `libtdjson.dylib` built from TDLib
  1.8.0 with compatibility patches.
- Includes TDLib cache-parse recovery for older or incompatible local caches.
- Includes basic fallback tiles for stickers that cannot render their native
  WEBP, TGS, or WEBM content on this legacy lane.

## Known Gaps

- This release is experimental and may show fallback UI where media rendering is
  incomplete.
- Some stickers, animations, videos, or document previews may still be limited.
- The app is unsigned and not notarized, so Mountain Lion may require a manual
  first launch flow.
- This is not the normal Mavericks release. Users on OS X 10.9.5 should prefer
  the regular Telegraphica `v0.4.5-beta` build unless they are intentionally
  testing the Mountain Lion branch.
