# Telegraphica v0.4.1-alpha.1 — First Public Alpha ✦

Telegraphica is an experimental native Telegram client for OS X 10.9.5
Mavericks on Intel `x86_64`. This alpha is intended for real legacy-Mac testing:
sign in, read chats, send messages, preview media, and collect feedback from
older Macs that modern Telegram clients no longer support.

## ✨ Highlights

- Adds an in-app first-run Telegram API setup screen when
  `~/Library/Application Support/Telegraphica/tdlib-config.plist` is missing.
- New users can now enter `api_id` and `api_hash` directly in Telegraphica,
  save them locally, and continue to phone/code/2FA login without Terminal
  setup.
- Existing local TDLib config files are preserved; only `api_id` and `api_hash`
  are updated when the setup form is saved.
- Bundles the Mavericks-compatible TDLib JSON library in the release app.
- Includes the current native chat, media, notification, profile, settings, and
  update-checking alpha feature set.

## 🚀 First Launch

1. Download the DMG or app zip.
2. Open Telegraphica with right-click / Control-click → **Open** if Gatekeeper
   warns.
3. If asked, enter your Telegram `api_id` and `api_hash` from
   `my.telegram.org`.
4. Continue with phone number, login code, and 2FA password if Telegram asks.

## 🔐 Security Notes

Telegram `api_id` and `api_hash` are stored only in the user's local
Application Support folder and must not be committed, uploaded, or included in
release assets. TDLib databases, sessions, login codes, 2FA passwords, and
Keychain material remain outside the repository and release package.

## ⚠️ Known Alpha Gaps

- Sticker and GIF rendering is still incomplete and can fall back to emoji.
- The app is unsigned/not notarized, so Gatekeeper may require right-click Open.
- Release confidence still comes from OS X 10.9.5 / Xcode 6.2 HITL testing.
