# Telegraphica v0.3.0-alpha.3 — Bundled Mavericks Installer Candidate

This prerelease is intended to be built on the OS X 10.9.5 Mavericks validation
machine with Xcode 6.2 and a known-working Mavericks-compatible TDLib JSON
library.

## Goal

- Build `Telegraphica.app` with `Contents/Frameworks/libtdjson.dylib` already
  bundled.
- Package the app into an HFS+ DMG that Mavericks can mount.
- Provide an app zip and SHA256 checksums alongside the DMG.
- Avoid asking end users to install CMake, MacPorts, OpenSSL, or TDLib.

## Expected Assets

- `Telegraphica-v0.3.0-alpha.3-macos10.9-x86_64.dmg` — drag-and-drop installer
  candidate for Mavericks.
- `Telegraphica-v0.3.0-alpha.3-macos10.9-x86_64.app.zip` — zipped app bundle.
- matching `.sha256` files.

## Build

On the legacy Mac:

```bash
./scripts/package_legacy_release_artifacts.sh --tdjson /path/to/libtdjson.dylib
```

The script rebuilds Telegraphica, validates and bundles TDLib, creates the
installer DMG/app zip, and writes release artifacts into `dist/`.

## Security Notes

Release assets must not include Telegram `api_id`, `api_hash`, login codes, 2FA
passwords, TDLib databases, session files, local Keychain material, or user
media caches.
