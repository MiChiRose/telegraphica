# Telegraphica v0.3.0-alpha.2 — Mavericks Installer Hotfix

This prerelease fixes the first public alpha installer image for OS X 10.9.5
Mavericks.

## What Changed

- Rebuilt the DMG installer as `UDZO` with an `HFS+` filesystem.
- Added an explicit `SPUD`/Apple partition layout for older macOS mounting.
- Bumped the app version to `0.3.0-alpha.2` and build number to `3002`.

## Why

The `v0.3.0-alpha.1` DMG was created on a modern macOS host without an explicit
filesystem. Modern `hdiutil` produced an APFS-backed image, and OS X Mavericks
cannot mount APFS images. Mavericks reported:

```text
no mountable file systems
```

## Assets

- `Telegraphica-v0.3.0-alpha.2-installer.dmg` — HFS+ drag-and-drop installer
  for Mavericks-era Macs.
- `Telegraphica-develop-<sha>-hfs-dmg-hotfix.zip` — source/build handoff archive
  for the legacy Mac flow.

## Known Limitations

- This is still an alpha prerelease.
- The DMG is not notarized.
- A compatible `libtdjson.dylib` is required for Telegram functionality.
- Sticker and GIF rendering remains incomplete and may use emoji fallbacks.

## Security Notes

Release assets do not include Telegram `api_id`, `api_hash`, login codes, 2FA
passwords, TDLib databases, session files, or local Keychain material.
