# Telegraphica v0.4.3-alpha.1 — Animated Stickers ✦

Telegraphica is an experimental native Telegram client for OS X 10.9.5
Mavericks on Intel `x86_64`. This alpha focuses on sticker playback and the
Mavericks-safe media path while preserving the public-alpha login, chat,
notification, profile, settings, voice-message, and release workflow.

## 🚀 What Should I Download?

If you just want to install Telegraphica, download this file from **Assets**:

👉 **`Telegraphica-v0.4.3-alpha.1-macos10.9-x86_64.dmg`**

Then open the DMG and drag **Telegraphica.app** into **Applications**.

The `.app.zip` file is an alternate app bundle package. The `.sha256` files are
for checksum verification. You do not need them to install Telegraphica.

## ✨ Highlights

- WEBP, TGS, and WEBM/VP9 stickers now render in chats on OS X Mavericks.
- Animated WEBM sticker playback uses a bundled Mavericks-compatible decoder
  instead of relying on AVFoundation.
- Sticker bubbles were tuned so animated and static stickers sit higher and
  cleaner inside their rounded frames.
- Recent stickers keep rounded thumbnails and safer fallback behavior when a
  format cannot be rendered.
- Component diagnostics were expanded for media playback debugging without
  exposing private file names in normal logs.
- Release builds still bundle TDLib and sanitized Telegraphica app credentials,
  while user sessions, phone numbers, login codes, and TDLib databases remain
  local to each Mac.

## 🔑 First Launch

1. Download the `.dmg` file.
2. Open it and drag **Telegraphica.app** into **Applications**.
3. If macOS blocks the app, right-click / Control-click it and choose **Open**.
4. Sign in with your Telegram phone number, login code, and 2FA password if
   needed.

## 🔐 Security Notes

The bundled config contains app-level Telegram API credentials only. User login
state, TDLib databases, phone numbers, login codes, 2FA passwords, and Keychain
material remain local to each user's Mac.

## ⚠️ Known Alpha Gaps

- Some sticker packs may still expose edge-case animation/layout bugs; please
  report the sticker format and a screenshot if that happens.
- The app is unsigned/not notarized, so Gatekeeper may require right-click Open.
- Release confidence still comes from OS X 10.9.5 / Xcode 6.2 HITL testing.
