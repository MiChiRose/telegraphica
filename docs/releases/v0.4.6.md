# Telegraphica v0.4.6 — Open Beta ✦

Telegraphica is an experimental native Telegram client for OS X 10.9.5
Mavericks on Intel `x86_64`. This release continues the open beta line with
drawer, folder, profile layout, and messaging polish validated on the legacy
Mac workflow.

## 🚀 What Should I Download?

If you just want to install Telegraphica, download this file from **Assets**:

👉 **`Telegraphica-v0.4.6-macos10.9-x86_64.dmg`**

Then open the DMG and drag **Telegraphica.app** into **Applications**.

The `.app.zip` file is an alternate app bundle package. The `.sha256` files are
for checksum verification. You do not need them to install Telegraphica.

## ✨ Highlights

- Chat folder drawer now loads TDLib folders, can be opened from app headers,
  and keeps Profile/Settings layout stable while toggling.
- Profile and Settings keep their content aligned when the drawer is opened or
  closed.
- Chat search, reply previews, document bubbles, and jump-to-latest navigation
  continue from the previous beta line.
- WEBP, TGS, and animated WEBM/VP9 stickers remain supported on Mavericks.
- Voice-message playback uses the bundled Mavericks-safe Ogg/Opus helper.
- Public builds fetch TDLib app configuration automatically on first launch.

## 🔑 First Launch

1. Download the `.dmg` file.
2. Open it and drag **Telegraphica.app** into **Applications**.
3. If macOS blocks the app, right-click / Control-click it and choose **Open**.
4. Sign in with your Telegram phone number, login code, and 2FA password if
   needed.

## 📦 Release Files

- `Telegraphica-v0.4.6-macos10.9-x86_64.dmg`
- `Telegraphica-v0.4.6-macos10.9-x86_64.app.zip`
- matching `.sha256` files.

## 🔐 Security Notes

Public builds do not ship user login state, TDLib databases, phone numbers,
login codes, 2FA passwords, or Keychain material. Those remain local to each
user's Mac.

## ⚠️ Known Beta Gaps

- The app is unsigned/not notarized, so Gatekeeper may require right-click Open.
- Some sticker packs or GIF-like media can still expose edge-case playback or
  layout issues.
- Release confidence still comes from OS X 10.9.5 / Xcode 6.2 HITL testing.
