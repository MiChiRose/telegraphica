# 🧩 Telegraphica v0.5.2 ✦ Unified Legacy Open Beta

Telegraphica v0.5.2 replaces the separate Mavericks and Mountain Lion release
lanes with one Intel `x86_64` application for OS X 10.8 through macOS 10.13.
The app selects its Mountain Lion-safe fallbacks at runtime, while Mavericks
and newer systems retain the normal feature path.

## What's New

- 🏔 One release artifact now targets OS X 10.8 Mountain Lion through macOS
  10.13 High Sierra.
- 🧩 The new Workshop provides optional downloadable games and utilities
  without bundling every module into the base application.
- 🎮 Workshop modules include Solitaire, Checkers, Minesweeper, Tic-Tac-Toe,
  Fifteen, Pac-Man, Diagnostic Center, and Media Workbench.
- 🧩 The unified host runs on OS X 10.8, while the first downloadable Workshop
  modules require OS X 10.9 or newer.
- 🪵 The Workshop uses its own Game Center-inspired visual surface, module
  filters, install/update states, progress feedback, and safe removal flow.
- 🧰 Xcode 5.1.1 and Xcode 6.2 legacy build paths now share the same source and
  packaging scripts.

## Compatibility And Stability

- Added Mountain Lion-safe application data, cache, Base64, TDLib storage, and
  reduced-background-work fallbacks.
- Preserved the regular Mavericks behavior when the same build runs on OS X
  10.9 or newer.
- Fixed a Mavericks startup crash caused by carrying a temporary `NSError`
  across a legacy manual-memory-management boundary.
- Temporary TDLib parameter or remote bootstrap failures remain recoverable
  instead of being reported as permanently missing configuration.
- The build keeps the generated Telegram connection provider and the bundled
  `libtdjson.dylib` across legacy rebuilds.
- Release validation now audits every Mach-O file in the application bundle for
  Intel `x86_64`, legacy load commands, portable dependencies, and a minimum OS
  version no newer than 10.8.

## Release Assets

Regular users should download:

`Telegraphica-v0.5.2-macos10.8-10.13-x86_64.dmg`

The release also provides an application ZIP and SHA-256 checksum files for
verification and troubleshooting.

Telegraphica remains an experimental, unofficial Telegram client. This unified
release was built with Xcode 5.1.1 and validated on real OS X 10.9.5 Mavericks
hardware. Native OS X 10.8 testing remains a separate compatibility check.
