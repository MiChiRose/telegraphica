# Telegraphica v0.3.0-alpha.1 — First Public Alpha

Telegraphica is an experimental, unofficial Telegram client for OS X 10.9.5
Mavericks on Intel Macs.

This first alpha is intended for real-world testing and feedback from legacy Mac
users. It is not a stable replacement for the official Telegram client yet, but
the core path is now usable enough to publish.

## Highlights

- Native Cocoa/AppKit interface for Mavericks-era Macs.
- TDLib authorization flow with phone, login code, and 2FA password.
- Chat list, unread counters, muted indicators, topics/supergroups, and message
  history.
- Text, photo-with-caption, reaction, and voice-message sending.
- Image grouping, image zoom preview, video playback, and document filename
  previews.
- Notification Center alerts, Dock unread badge, notification sound, and click
  through to the source chat.
- Typing indicators when TDLib reports active chat actions.
- Profile, settings, themes, language selector, logs, and about panel.

## Assets

- `Telegraphica-v0.3.0-alpha.1-installer.dmg` — drag-and-drop app installer.
- `Telegraphica-develop-<sha>-alpha-release.zip` — source/build handoff archive
  for the legacy Mac flow.

## Known Limitations

- This is a prerelease alpha.
- Sticker and GIF rendering is still incomplete and may use emoji fallbacks.
- The DMG is not notarized.
- A compatible `libtdjson.dylib` is required. Source archives do not vendor
  Telegram credentials, TDLib databases, sessions, or built TDLib binaries.
- Final compatibility confidence still comes from OS X 10.9.5 / Xcode 6.2 HITL.

## Security Notes

Do not upload Telegram `api_id`, `api_hash`, login codes, 2FA passwords, TDLib
databases, session files, or local Keychain material. Telegraphica keeps those
outside the repository and release archives.
