# Test fixtures

- `libwebp-test.webp` exercises the Mavericks WebP decoder path.
- `tgs-animation.json` is the `bounching_ball.json` rlottie example used to
  verify distinct rendered frames and Telegraphica's gzip-compressed TGS view.
  It comes from the pinned Samsung rlottie source and is covered by the
  permissive notices in `Vendor/rlottie/licenses`.
